    1. What is Semantic HTML?
     HTML that gives meaning to the content, like nav, section, h1-h6, etc.
    2. What is HTML used for? 
     To give structure to our webpage.
    3. What is an attribute and where do we put it?
     Give us more information about elements, such as the 'href' attribute in the <a> tag.
    4. What is the h1 tag used for? How many times should I use it on a page?
    For the most important heading on a page, only once.
    5. Name two tags that have required attributes
    <input> <textarea>
    6. What do we put in the head of our HTML document? 
     The metadata of an HTML document including tags such as title, style, meta, link, script, and base
    7. What is an id? 
     An attribute for an element to be used with HTML/CSS/JS
    8. What elements can I add an id to? 
      Every element
    9. How many times can I use the same id on a page? 
      Once
    10. What is a class? 
      An attribute for a group of elements to be used with HTML/CSS/JS
    11. What elements can I add a class to? 
      Every element
    12. How many times can I use the same class on a page?
      Multiple times 
    13. How do I get my link to open in a new tab?
      <a href="linkhere" target="_blank"
    14. What is the alt attribute used for? 
      To specify an element for screenreaders, or if it doesn't display
    15. How do I reference an id?
      # {}
    16. What is the difference between a section and a div
     A section is a semantic tag, and a div is not
    17. What is CSS used for? 
      To style a webpage
    18. How to we select an element? Example - every h2 on the page
      h2 {}
    19. What is the difference between a class and an id? - Give me an example of when I might use each one
      An id is for one elements such as <h1 id="myHeader">, a class is for groups of elements such as <div class="container"><h2><p><img></div

      .classname {}
    21. How do we select a p element with a single class of “human””?
      p.human {}
    22. What is a parent child selector? When would this be useful? 
      Selects all child elements of the parent element. When needing to style certain elements
    23. How do you select all links within a div with the class of sidebar?
      a.sidebar {}
    24. What is a pseudo selector?
      A selector used to modify the state of elements
    25. What do we use the change the spacing between lines?

      Used in many companies as a way so everyone can contribute in teams.
    35. How often should I commit to github?
      When making significant changes.
    36. What is the command we would use to push our repo up to github? 
      push -u origin branch-name
    37. Walk me through Lambda's git flow. 
      1.) Fork the project repo
      2.) Check it's actually forked
      3.) Add access to TL in settings
      4.) Copy repo link
      5.) Change into directory where repo will be cloned
      6.) git clone repo link to clone repo to machine
      7.) go into the repo with cd repo name
      8.) create own branch with git checkout -b firstname-lastname
      9.) Work on the repo
      10.) use git add . , git commit -m "msghere" , git push -u origin branch-name to commit changes and push to github
      11.) Submit a pull request but first verify you selected your name-branch
      12.) click new pull request with your name-branch
      13.) In PR page change base repo to the forked repo when submitting PR
      14.) Add the TL to review and submit the PR
      15.) Don't merge your own pull request, the TL will take care of that
